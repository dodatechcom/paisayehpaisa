# Project Overview

## Domain
paisayehpaisa.com

## Tagline
Crypto in plain language.

## Mission
Answer the questions BitcoinTalk users ask every day — but in one place, with simple language, visual guides, and interactive tools.

## Why this exists
BitcoinTalk has 67M+ posts. The same beginner questions repeat daily:
- "Which wallet should I use?"
- "How do I buy Bitcoin?"
- "I got scammed, help"
- "What is a seed phrase?"
- "What is DCA?"

Answers are buried in 50-page threads. No authority signal. No structured learning path.

## Content format
- Articles (markdown → static HTML)
- Interactive tools (vanilla JS)
- Comments via giscus (GitHub Discussions)

## Hosting
GitHub Pages / Netlify — free, no inode limits

## Framework
Astro (static site generator)

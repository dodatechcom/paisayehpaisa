Hot wallets are software wallets that are connected to the internet. They're convenient for daily use, DeFi, and small amounts. The choice between mobile and desktop depends on your use case and security needs.

## Mobile Wallets

**Examples:** Trust Wallet, MetaMask Mobile, Phantom Mobile, Coinbase Wallet, Exodus Mobile

| Pros | Cons |
|---|---|
| Convenient — use anywhere | Phone is easily lost/stolen |
| QR code scanning for payments | Smaller screen for complex dApps |
| Biometric security (Face ID, fingerprint) | Phones are harder to secure |
| Push notifications for transactions | Mobile OS may auto-upload photos (seed phrase risk) |
| NFC payments | App Store restrictions |

### Best For:
- Small amounts (<$1,000)
- Daily spending and payments
- Quick portfolio checking
- QR code payments

## Desktop Wallets

**Examples:** MetaMask (browser), Rabby, Exodus Desktop, Electrum, Guarda

| Pros | Cons |
|---|---|
| More secure (harder to steal laptop than phone) | Less portable |
| Full dApp browser | Requires computer to use |
| Better for complex DeFi transactions | Computer may have malware |
| Can use with hardware wallets | No biometric security |
| Open source, auditable | |

### Best For:
- Medium amounts ($1,000-$10,000)
- DeFi interactions
- NFT management
- Trading on DEXs
- Multi-chain management

## Security Comparison

| Aspect | Mobile | Desktop |
|---|---|---|
| Malware risk | Lower (app store vetting) | Higher |
| Theft risk | Higher (phone easily stolen) | Lower |
| Seed phrase exposure | Risk (screenshots, cloud backup) | Risk (keyloggers) |
| OS security | iOS = strong, Android = variable | Windows = weak, macOS/Linux = stronger |
| Physical access | Phone PIN > computer password | Usually weaker |

## The Right Hot Wallet for Each Use

| Amount | Recommended |
|---|---|
| <$500 | Mobile wallet (Trust Wallet, Phantom Mobile) |
| $500-$5,000 | Desktop wallet (MetaMask, Rabby) |
| $5,000-$50,000 | Desktop + hardware wallet (Ledger + MetaMask) |
| >$50,000 | Hardware wallet only |

## Two-Wallet Strategy

Use both mobile and desktop for different purposes:

**Mobile wallet:**
- Small amount for daily spending
- QR code payments
- Track portfolio on the go

**Desktop wallet:**
- DeFi interactions
- Larger transactions
- Connected to hardware wallet

**Never put the same seed phrase on both mobile and desktop.** If one device is compromised, the other is safe.

## Verdict
Mobile wallets are more convenient for daily use. Desktop wallets are more secure and better for DeFi. For most people, use a mobile wallet for small spending and a desktop wallet with a hardware wallet for significant holdings. Never use the same seed phrase on both.

**Question from BitcoinTalk:** "What's the best way to track my crypto portfolio across multiple exchanges and wallets?"

**Short answer:** Use Zerion (best for quick checking) or CoinTracker (best for tax tracking). Both connect to exchanges via API and support multiple blockchains.

## What to Look For in a Tracker

| Feature | Why It Matters |
|---|---|
| Multi-chain support | Your crypto is probably on 2-4 different chains |
| Exchange API integration | Track what's on Coinbase, Binance, Kraken |
| Wallet connection | Read-only access to your wallets |
| Tax reporting | Generate tax forms automatically |
| Real-time prices | Accurate portfolio value |
| Transaction history | See all your trades in one place |

## Best Overall: Zerion

Zerion connects to your wallets (MetaMask, Ledger, etc.) via read-only API and shows your full portfolio across all chains.

**Supported:** Ethereum, Arbitrum, Base, Optimism, Polygon, Solana, and 10+ more.

**Features:**
- Real-time portfolio value
- Transaction history
- DeFi positions (lending, liquidity pools, staking)
- NFT gallery
- Web and mobile apps

**Price:** Free for basic tracking. $45/year for premium (notifications, advanced analytics).

**Best for:** Most people who want a simple dashboard of everything they own.

## Best for Tax: CoinTracker

CoinTracker connects to exchanges (Coinbase, Binance, Kraken) via API and to wallets via read-only access. It automatically categorizes transactions and calculates gains/losses.

**Features:**
- Import from 300+ exchanges and wallets
- Automatic cost basis calculation (FIFO, LIFO, specific ID)
- Tax forms (8949, Schedule D)
- Portfolio tracking dashboard

**Price:**
- Free: Up to 25 transactions
- Pro: $59/year (1,000 transactions)
- Premium: $199/year (10,000 transactions)

**Best for:** US users who need tax reporting alongside portfolio tracking.

## How to Set Up a Portfolio Tracker

### Via Wallet Connection (Read-Only)
1. Go to Zerion or Zapper
2. Click "Connect Wallet"
3. Select "Watch only" or "Read-only" mode
4. Enter your wallet address
5. The app shows all tokens across all chains

**Safety:** Read-only mode means the app can only view — it cannot initiate transactions. Your funds are safe.

### Via Exchange API
1. Go to CoinTracker or Koinly
2. Select your exchange (Coinbase, Binance, Kraken)
3. Generate API keys on the exchange (enable "Read" only — never enable "Trade" or "Withdraw")
4. Paste the keys into the tracker
5. Transactions import automatically

## Verdict

For most people: **Zerion** for daily portfolio tracking (free, connects to wallets, shows everything). **CoinTracker** or **Koinly** for tax season (imports from exchanges, calculates gains).

Track your portfolio monthly. Review your allocation quarterly. Rebalance if needed. Don't check daily — it leads to emotional trading.
